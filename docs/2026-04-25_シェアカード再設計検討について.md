# シェアカード再設計検討について

- 日付: 2026-04-25
- 対象: LOL-MBTI（サモナー診断サイト）
- 背景: 既存シェアカードのXタイムラインでのインパクトが弱く、共有誘引力に課題。レイアウトを根本から再考する流れで、Xの表示仕様調査→3案モック→案絞り込み→2段階差別化→情報量増の方針検討まで進めた

## 共有したくなるカードの原則（議論ベース）

人がシェアカードを共有する心理メカニズム:

1. 自己表現 — 一枚で「これが自分」と言えるアイデンティティ性
2. 自慢・特別感 — 希少・強そう・カッコいいと感じる勲章性
3. 会話フック — 「お前は何タイプ？」と聞きたくなる比較軸
4. ひと目で読める — TLでスクロールされても主要要素が伝わる
5. スクショよりキレイ — 保存したくなる質感

現状カードの課題感:
- 情報密度は豊富だが「主役」が不明確（全セクションが等しい重みで並ぶ）
- TLで縮小表示時に何も伝わらない可能性
- チャンピオンスプラッシュアートを活用できておらず、ビジュアルインパクトが弱い
- サイト固有資産である「サモナータイプ名」が小さく扱われている

## ユーザーの方向性確認

- インパクト弱め＝改善したい
- 情報量は維持したい（削りたくない）
- メインターゲットはLoLプレイヤーだが、未プレイ層にも刺したい

→ 「サムネで撃ち抜く × 情報密度維持」の二重要件を満たす設計が必要

## Xタイムライン画像表示仕様（2026時点・調査結果）

### 確定情報
| 条件 | 挙動 |
|------|------|
| 2:1 〜 1:1 | タイムラインで切り取られない |
| 16:9 (1200×675) | 推奨。切り取りなし |
| 1:1 (1200×1200) | 切り取りなし |
| 4:5 (1080×1350) | 「縦に大きく表示」派と「1.91:1にセンタークロップ」派で情報源が割れる |
| 3:4 より縦長（9:16等） | タイムラインで中央部分のみ表示。全体はタップ後 |

### 補足
- クロップは中央基準（上下両方が切れる、上だけではない）
- ファイルサイズ制限: モバイル経由 5MB / Web 15MB
- 4:5 の挙動は X のアプリバージョン・OS・時期で変動している可能性が高く、保険をかけた設計が安全

### 情報源
- HeyOrca, Tweet Archivist, Image for Post, posteverywhere, Hootsuite ほか複数の2026年版ガイド

## レイアウト3案比較

| | 案1: 16:9 横長 | 案2: 1:1 正方形 | 案3: 4:5 縦長 |
|---|---|---|---|
| サイズ | 1200×675 | 1200×1200 | 1080×1350 |
| クロップ | 切り取りなし | 切り取りなし | 不確実（センタークロップの可能性） |
| 情報量 | 最小（ヒーローのみ） | 中（ヒーロー + 信条） | 大（現行と同等） |
| インパクト | 最強 | 程よい | 縦長で目立つが赤枠リスク |

→ ユーザーは **案2（1:1）** を選定

## 案2 採用後の設計議論

### Summoner名表示の追加

ユーザー意見: 自己紹介カードのように「自分の診断だ」と感じる愛着が、共有意欲を上げるはず

メカニズム:
- 名前 = 所有感の錨。「a diagnosis」→「MY diagnosis」に変える
- 16personalities, ポケモン個人ID, LINEスタンプの「〇〇が送った」と同じ原理
- ただしバランス重要: でかすぎ→自己紹介スライド化、小さすぎ→愛着フック機能せず

### 入力フローと制約（決定事項）

- 入力タイミング: ダウンロードボタン押下時にモーダル（ルートC、Just-in-time）
- 文字種: 英数字 + 日本語のみ
- 文字数上限: 10字
- 永続化: 不要（毎回入力）
- 表示位置: タイプ名の上、装飾ライン付きの「署名」ポジション
- ラベル: 「Summoner」プレフィックス不要、名前のみ表示

### 2段階差別化の方針

コードを確認した実態:
- Phase 1（16Q完了）: `is25` true → タイプ名・サブタイトル表示
- Phase 2（32Q完了）: `is40` true → 信条・S/W・レーダー表示

差別化戦略（推奨ベース）:
- **Stage 1（16Q）= 身分証カード**: 名前 + タイプ + チャンピオン + % + レーン。信条なし、装飾最小
- **Stage 2（32Q）= 叙任カード**: 上記 + 信条 + 二重金縁 + 四隅角飾り + 背景グロウ + "PHASE II" 表記

理由:
- 信条は最強のシェア燃料（論理より感情がリポストされやすい）
- S/W・レーダーはサイト側の結果ページに集約（カードは「看板」、サイトが「中身」）
- Stage 2 の差を「ご褒美」化することで「全問やる動機」を視覚化

## 表示項目インベントリ

| 項目 | 出典 | カード適合性 |
|------|------|------|
| ユーザー名 | 新規入力 | ◎ 愛着フック |
| チャンピオン画像/名前/タイトル/% | 既存 | ◎ 必須 |
| レーン | 既存 | ◎ 必須（サイドバッジ） |
| タイプ名・サブタイトル | 16Q+ | ◎ 主役 |
| 信条（Catchphrase） | 32Q | ◎ 1:1の決めゼリフ |
| 強み・弱み | 32Q | △ 分析的で空気を冷やす |
| レーダーチャート | 32Q | △ 1:1だと小さくなりすぎ |
| 相性タイプ・相性チャンピオン | 32Q | ✗ 説明が必要 |
| 味方／敵から見たあなた | 32Q | ✗ 長文 |

## 情報量強化のルート検討（議論中）

ユーザー要望: 情報をもっと増やしたい

提示した3ルート:

### ルートA: 1:1 のまま詰め込む
- 追加可能: S/W、TOP3ステータスバー
- 長所: サムネインパクト維持、クロップ無縁
- 短所: 文字が小さくなる、詰めすぎるとチラシ感

### ルートB: 4:5 縦長に切替
- 追加可能: S/W + レーダー + 信条すべて
- 長所: 情報密度1.5倍
- 短所: Xでセンタークロップされる可能性（赤枠問題復活）

### ルートC: 2枚画像ポスト（表 + 裏）★推奨
- 1枚目: 1:1の身分証カード（サムネで勝つ）
- 2枚目: 1:1の裏面カード（レーダー + S/W + 相性タイプ + 相性チャンピオン全部入り）
- 長所:
  - サムネインパクト維持しつつ情報量ほぼ無制限
  - スワイプの楽しみ＝エンゲージメント増
  - 診断サービスの定番パターン
- 短所:
  - canvas 2枚レンダリングの実装増
  - 2枚目を見ないユーザーがいる（1枚目で完結すれば問題なし）
